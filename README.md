# Media Converter Project

## Features
- Convert Video to MP3
- Convert Audio to Video (AI-style waveform visualization)

## Setup
```bash
pip install -r requirements.txt
```

## Usage
1. Put a video file as input/sample.mp4
2. Run:
```bash
python video_to_mp3.py
```

3. Put an audio file as input/sample.mp3
4. Run:
```bash
python audio_to_video.py
```
